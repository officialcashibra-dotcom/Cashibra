What is Cashibra?
Cashibra is a fun learning project where kids and teens learn money, skills, and financial basics through games and challenges.
Is this website real or still in development?
This is a demo/testing version. More features like battles, points, and missions will be added later.
Is Cashibra safe?
Yes, the website does not collect any sensitive data. Everything here is only for education and testing.
Will I really earn money?
No real money is given. This is a learning and demo website to teach financial skills in a fun way.
Who is the creator?
Cashibra was created by a 12 year old learner (Aarav Mahule) who wanted to teach financial skills to other teens.
